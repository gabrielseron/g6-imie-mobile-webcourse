# Cericoil
##### IOS & Android webcourse app


## Dev Instructions

Install dependencies

```npm i```


Run Android app

```ionic capacitor add android```

```npx jetify```

```npx cap sync android```

```ionic capacitor run android -l --external```

Run Ios app

```ionic capacitor add ios```

```npx jetify```

```npx cap sync ios```

```ionic capacitor run ios -l --external```
